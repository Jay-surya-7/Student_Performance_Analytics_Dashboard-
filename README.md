## ğŸ“Š Dashboard Features

Based on the current dashboard:

- **KPI / Summary View**
  - Displays total students, average marks, and overall pass percentage.

- **Subject Average vs Overall Average**
  - Compares subject-wise average marks with the overall average marks.

- **Average Marks by Gender**
  - Bar chart showing average marks for:
    - Male
    - Female
    - Non-Binary

- **Pass vs Fail Distribution**
  - Donut chart showing the count and percentage of:
    - Pass students (~57%)
    - Fail students (~43%)

- **Pass Percentage by Year**
  - Line chart showing pass percentage trend from **2018 to 2025**
  - Highlights year-wise fluctuations and overall trend in results.

- **Filters / Slicers**
  - Enables interactive exploration of the data (as visible in the report UI).
